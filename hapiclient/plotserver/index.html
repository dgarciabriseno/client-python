Plot data from a HAPI server

Usage

Return a PNG image of PARAMETERS plotted over the timerange START/STOP. At present, PARAMETERS can only be a single parameter.

?server=HAPIURL&id=ID&parameters=PARAMETERS&time.max=START&time.min=STOP

Open a ViViz gallery

?server=HAPIURL&id=ID&format=gallery

Other parameters

  • format [png], svg, or pdf
  • transparent [true] or false
  • usecache [true] or false
  • figsize [7,3] image size in inches
  • dpi [144] dpi of image
